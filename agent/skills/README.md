# Agent Skills

Skills from https://ethskills.com used by the Cocoa Ledger agent.

## Included Skills

- `standards.md` — Ethereum token standards (ERC-20, ERC-721, ERC-1155, ERC-8004)
- `security.md` — Smart contract security patterns and audit checklist

## How to Add Skills

1. Visit https://ethskills.com
2. Find the skill you need
3. Download or copy the content
4. Save as `skills/<name>.md`
5. Reference in the agent code or AGENTS.md

## Usage

The agent uses these skills as reference material when analyzing blockchain data and generating quality assessments. Skills provide context about token standards and security best practices.
